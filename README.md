## A Simple Ebay Website where the user can: 

- Signup 
- Login 
- Create/Edit Listings
- Comment on Listings
- End/Win Auctions
- Add/Remove items to Watchlist
- Bid on items in Active Listings

## Languages/addons used:
- Python
- Html5
- CSS3
- Bootstrap
- SqlLite
